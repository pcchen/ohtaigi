# tshia 車
(白) tshia 車
(文) ki 車, ku 車  

```{dropdown} 教
1. {{名}} 陸地上靠輪子轉動而運行的交通工具。例：火車　hué-tshia、計程車　kè-thîng-tshia。
1. {{動}} 以車子搬運東西。例：車砂　tshia sua(搬運砂石)、車糞埽　tshia pùn-sò(清運垃圾)。
```
